### List all tasks: `list`
Format: `list`  
Lists out all existing tasks in the task list in the format:  
```
1) [T][ ] todo
2) [E][ ] event (at: TIME)
3) [D][ ] deadline (by: TIME)
```
The first square ([ ]) denotes what type of task it is:
- [T]: todo 
- [E]: event 
- [D]: deadline 

The second square ([ ]) denotes the task status:
- [X]: task completed
- [ ]: task not completed
